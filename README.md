# sillypass

## Installation

`go get github.com/btoll/sillypass`

## CLI Usage

    Option | Description
    ------------ | -------------
    -n=NUM | The length of the password (defaults to 12 characters).
    -h, --help | Display help.

## Scripting Usage

    Command | Description
    ------------ | -------------
    generate(n) | Generate a password of length `n`.
